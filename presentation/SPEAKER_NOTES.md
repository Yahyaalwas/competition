## Transition Sentences

| From → To | Bridge |
|---|---|
| Title → Problem | *"But first — why does this problem matter?"* |
| Problem → Why It Matters | *"The numbers describe the problem. The impact explains the urgency."* |
| Why → Platform | *"Here's how we solved it."* |
| Platform → Architecture | *"Let me show you what's behind the interface."* |
| Architecture → Data | *"Starting with the data foundation."* |
| Data → Intelligence | *"But forecasting is only half the story."* |
| Intelligence → Scenario | *"Now let's talk about what policymakers can do with that intelligence."* |
| Scenario → Explainability | *"Every AI system used for government decisions must earn trust."* |
| Explainability → UX | *"Trust is communicated through design, not just documentation."* |
| UX → Impact | *"Who does this platform serve, and how?"* |
| Impact → Conclusion | *"Let me bring this together."* |

## Presenter Tips

| Do | Don't |
|---|---|
| Say "policymakers", "ministers", "strategic planners" | Say "users" |
| Pause after showing Arabic content | Rush past bilingual features |
| Quantify: "2.5 percentage points", "#4 to #2" | Say "the model improved" |
| Let judges read the executive summary | Talk over the text |
| Acknowledge limitations ("heuristic elasticities, not econometric") | Claim perfection |
| End with "60 million people" | End with features |
| Advance slides confidently on schedule | Wait for questions mid-slide |
